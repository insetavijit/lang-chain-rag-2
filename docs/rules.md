### Exceptions

The following markdown files may exist at the project root:

| File | Location | Reason |
|------|----------|--------|
| `README.md` | `/` (root) | Standard project documentation |
| `CHANGELOG.md` | `/` (root) | Version history (if needed) |
| `CONTRIBUTING.md` | `/` (root) | Contribution guidelines (if needed) |
| `LICENSE` | `/` (root) | License file |


### RULEs
1. Create all .md files under docs dir
